## Program:
```
import numpy as np

def find_rank(matrix):
    return np.linalg.matrix_rank(matrix)

matrix =  [[5,-3,-10],[2,2,-3],[-3,-1,5]]
rank = find_rank(matrix)
print(rank)

```
